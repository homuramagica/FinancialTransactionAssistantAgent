# 🚨 Anthropic의 Mythos는 사이버 보안 도구이자 공격 무기 후보가 됐습니다

&nbsp;

Anthropic이 새 모델 Mythos를 일반 공개 대신 엄선된 파트너에게만 제한 배포하기로 한 것은 이 도구가 제로데이를 찾아내는 보안 자동화 엔진일 뿐 아니라, 잘못 풀리면 핵심 인프라와 기업 네트워크를 뚫는 공격 무기까지 될 수 있다는 점을 스스로 인정한 결정입니다.

&nbsp;

⚡ **중요한 이유:** AI가 사이버 보안을 돕는 수준을 넘어 공격자와 방어자 모두에게 초고속 취약점 발굴 능력을 주기 시작했다는 점이 이번 사건의 핵심입니다.
- Anthropic은 Mythos가 소프트웨어와 컴퓨터 시스템의 취약점을 너무 잘 찾아내기 때문에 일반 대중에 공개하지 않겠다고 밝혔습니다.
- 회사 설명에 따르면 Mythos는 테스트 과정에서 모든 주요 운영체제와 주요 웹브라우저에서 수천 개의 제로데이를 발견했습니다.
- 더 큰 문제는 모델 공개 계획이 처음 발표된 바로 그날 비공개 온라인 포럼의 무단 사용자들이 접근권을 얻었다는 점입니다.

&nbsp;

🧬 **작동 원리:** Mythos는 사람이 일일이 붙지 않아도 취약점 탐색과 익스플로잇 조합을 자동으로 밀어붙이는 쪽으로 진화했습니다.
- Anthropic은 이 모델이 수십 년 동안 사람 검토와 자동 보안 테스트를 견딘 취약점도 찾아냈다고 주장했습니다.
- 예로 27년 된 OpenBSD 결함과 리눅스 커널 취약점 연쇄를 찾아 완전한 시스템 장악 경로를 만들었다고 설명했습니다.
- 비전문가가 원격 장악 방법을 요청하고 밤새 돌려놓은 뒤 아침에 완성형 익스플로잇을 받았다는 내부 사례도 제시됐습니다.

&nbsp;

🛡 **현재 상황:** Anthropic은 Project Glasswing라는 방어용 통제 프로그램으로 위험을 묶어두려 하지만 아직 완성형 안전장치는 아닙니다.
- 참여 기관에는 아마존(AMZN), 애플(AAPL), 구글(Alphabet, GOOGL), 마이크로소프트(MSFT), 엔비디아(NVDA), 팔로알토네트웍스(PANW), 크라우드스트라이크(CRWD), 브로드컴(AVGO), 시스코(CSCO), JPMorgan Chase(JPM), 리눅스 재단이 포함됩니다.
- 이들은 Mythos를 침투 테스트와 방어 보안 작업에 써서 더 많은 취약점을 더 빨리 찾는 실험을 하게 됩니다.
- Anthropic도 드물지만 이상하게 행동하거나 우려스러운 행동을 보인 적이 있다고 인정해, 통제 장치가 아직 진행형임을 시사했습니다.

&nbsp;

🧨 **숨은 의미:** 보안 AI 경쟁은 이미 Anthropic만의 이야기가 아니어서, 앞으로는 통제 실패가 개별 회사 리스크를 넘어 시장과 국가 안보 리스크가 될 수 있습니다.
- 블룸버그는 OpenAI의 Codex Security와 구글의 Big Sleep agent도 취약점 탐지용으로 개발되고 있다고 전했습니다.
- 이스라엘 보안 스타트업 Buzz는 다섯 개 AI 에이전트를 묶어 알려진 취약점을 98% 성공률로 악용하는 도구를 만들었다고 주장합니다.
- 일리노이대의 강 왕 교수처럼 외부 연구자들은 아직 Anthropic의 성능 주장을 독립 검증하지 못했다고 말하고 있어, 과장과 실제 위협을 함께 가려보는 단계가 막 시작됐습니다.
