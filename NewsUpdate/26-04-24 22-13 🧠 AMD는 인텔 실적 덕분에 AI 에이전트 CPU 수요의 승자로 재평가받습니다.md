# 🧠 AMD는 인텔 실적 덕분에 AI 에이전트 CPU 수요의 승자로 재평가받습니다

&nbsp;

Intel의 블록버스터 실적이 CPU 시장의 구조적 회복 신호로 해석되면서, Advanced Micro Devices(AMD)는 자체 뉴스 없이도 AI 에이전트 시대 서버 CPU 수요의 더 큰 수혜주로 다시 평가받고 있습니다.

&nbsp;

🧩 **중요한 이유:** AI 투자 서사는 GPU 훈련 수요에서 추론과 에이전트 실행으로 넓어지고 있고, 이 변화는 한동안 조용했던 CPU 시장을 다시 핵심 병목으로 만들고 있습니다.
* Intel 실적 발표 뒤 AMD 주가는 금요일 프리마켓에서 12% 올라 340.46달러를 기록했습니다.
* Arm Holdings(ARM)도 8.5% 상승해 222.05달러를 보였습니다.
* AMD 자체의 별도 뉴스는 없었습니다.
* 투자자들은 Intel의 실적을 CPU 수요 회복의 선행 신호로 해석했습니다.

&nbsp;

📊 **현재 상황:** D.A. Davidson은 Intel 실적을 근거로 AMD 투자의견을 상향했습니다.
* 애널리스트 Gil Luria는 AMD를 Neutral에서 Buy로 올렸습니다.
* 목표주가는 220달러에서 375달러로 크게 높였습니다.
* 그는 한때 졸린 시장이던 CPU 시장이 큰 단계 상승을 앞두고 있다고 봤습니다.
* Intel 주가도 같은 실적 발표 뒤 28% 급등했습니다.

&nbsp;

⚙️ **작동 원리:** AI 워크로드가 대형언어모델 훈련에서 모델을 이용해 작업을 수행하는 추론 단계로 옮겨가며 CPU의 역할이 커지고 있습니다.
* 업계는 이 변화를 agentic AI workload라고 부릅니다.
* 추론과 에이전트 실행에는 여러 작업을 조율하고 서버 환경을 운용하는 CPU가 필요합니다.
* AMD는 Intel의 핵심 CPU 경쟁자입니다.
* Arm은 다른 회사용 칩 설계로 알려져 있지만 지난달 자체 CPU 생산 계획을 발표했습니다.

&nbsp;

🏁 **반대편 시각:** Intel이 헤드라인을 차지했지만 일부 애널리스트는 AMD가 더 좋은 CPU 투자처라고 봅니다.
* Jefferies는 Intel 목표주가를 올리면서도 AMD에 더 큰 기대를 표시했습니다.
* Jefferies는 Intel이 2027년까지 이어지는 두 자릿수 서버 유닛 성장 모멘텀을 언급했지만, AMD가 더 나은 성장을 볼 수 있다고 썼습니다.
* Jefferies는 AMD의 신형 Venice 칩이 올해 말이나 내년 초 나올 것으로 기대합니다.
* 해당 팀은 AMD에 Buy 등급과 300달러 목표주가를 유지했습니다.

&nbsp;

📌 **결론:** AMD 랠리는 AI 인프라 사이클이 GPU 한 축에서 CPU, 추론, 에이전트 실행 인프라로 넓어지고 있음을 보여줍니다.
* Intel 실적은 CPU 시장 전체의 수요 탄력을 확인하는 촉매가 됐습니다.
* AMD는 Intel 회복의 경쟁자이면서 동시에 같은 구조적 수요의 수혜주로 읽힙니다.
* AI 반도체 투자에서는 학습용 GPU뿐 아니라 서버 CPU 공급과 가격 결정력도 더 중요해졌습니다.
