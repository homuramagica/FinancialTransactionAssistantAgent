# 🚇 뉴욕-뉴저지 Gateway 터널은 허드슨강 굴착 계약으로 큰 고비를 넘었습니다

&nbsp;

뉴욕과 뉴저지를 잇는 160억달러 규모 Gateway 철도 터널 프로젝트가 허드슨강 아래 새 터널 굴착을 맡길 12억9000만달러 건설 계약을 승인하며 미국 최대 인프라 사업 중 하나의 핵심 단계에 들어갔습니다.

&nbsp;

🏗️ **주요 뉴스:** Gateway Development Commission 이사회는 월요일 Traylor/Walsh/Skanska JV를 새 터널 굴착 계약자로 승인했습니다.

- 계약 규모는 12억9000만달러로, 원화로는 약 1조9,002억원입니다.
- 이 컨소시엄은 새 터널을 파낼 터널 보링머신을 감독하게 됩니다.
- Gateway의 프로그램 전달 책임자 James Starace는 이 회사가 복잡한 프로젝트를 수행하는 데 필요한 전문성을 갖췄다고 말했습니다.

&nbsp;

🚆 **배경 설명:** Gateway 계획은 허드슨강 아래 새 2선 철도 터널을 짓고 기존 터널도 보수하는 사업입니다.

- 기존 터널은 1910년에 개통됐고, Amtrak과 New Jersey Transit 열차가 뉴저지와 맨해튼을 오가는 핵심 통로입니다.
- 새 터널은 New Jersey와 뉴욕 Penn Station을 잇는 통근·도시간 철도 병목을 완화하기 위한 사업입니다.
- 전체 사업비는 160억달러, 원화로는 약 23조5,680억원 규모입니다.

&nbsp;

🔍 **무대 뒤 이야기:** 계약자 선정에는 여러 기관의 기술 전문가가 참여했습니다.

- 평가 패널에는 Gateway, New Jersey, New York, New Jersey Transit, Amtrak, Port Authority of New York and New Jersey의 전문가들이 들어갔습니다.
- 패널은 제안서를 낸 세 곳의 계약 회사를 검토했습니다.
- Starace는 이처럼 독특하게 어려운 프로젝트를 맡을 자원과 기술 전문성을 갖춘 회사가 많지 않다고 말했습니다.

&nbsp;

⚠️ **현실 점검:** 프로젝트는 연방 자금 문제와 Trump 행정부의 계약 검토라는 정치·규제 리스크도 겪고 있습니다.

- Gateway 관계자들은 2월에 연방 자금 지급이 일시 중단되면서 2주 동안 작업을 멈춰야 했습니다.
- 자금 지급은 재개됐지만, Trump 행정부는 Gateway 계약들이 새 규칙을 따르는지 계속 검토하고 있습니다.
- 해당 규칙은 인종 또는 성별에 기반한 계약 요건을 금지합니다.

&nbsp;

📌 **결론:** 이번 계약 승인은 Gateway 터널이 계획과 정치 논쟁의 단계를 넘어 실제 굴착 단계로 들어가는 신호입니다.

- 미국 동북부 철도망의 병목을 푸는 장기 인프라 투자라는 점에서 경제적 의미가 큽니다.
- 다만 연방 자금과 계약 규정 검토가 계속 남아 있어, 공사 속도는 기술보다 정책 리스크에 다시 흔들릴 수 있습니다.
