# Global Financial Crisis Writing Folder

- purpose: `2008년 금융위기`를 중심으로 한 장기 집필 프로젝트의 원고 저장 폴더
- audience: 일반 개인투자자도 재미있게 읽을 수 있는 교양서/연재물/에세이 독자

## 이 폴더에 저장하는 것

- 책 전체 목차
- 서문, 프롤로그, 에필로그 초안
- 장별 시놉시스
- 본문 챕터 초안
- 문장 다듬기 전 단계의 작업 원고
- 구성 메모, 브리지 문단 메모, 장 전환 메모

## 이 폴더에 저장하지 않는 것

- raw article 원문 아카이브
- 기사 수집용 상태 파일
- 검색용 SQLite DB
- 단순 조사 로그

위 자료들은 `/Research/`와 `Research/research_archive.sqlite3`에서 관리한다.

## 프로젝트 범위

이 폴더는 아래 흐름을 하나의 긴 서사로 묶는 집필 프로젝트를 다룬다.

- 서브프라임과 주택버블
- MBS, CDO, CDS, 신용평가, 레버리지
- 베어스턴스, 메릴린치, AIG, 리먼 브러더스
- 머니마켓펀드, 상업어음, 연준 유동성 프로그램
- TARP, 스트레스 테스트, 도드-프랭크법, 바젤3
- 오늘의 사모신용과 그림자금융

## 편집 원칙

- 설명은 쉽고 서사는 강하게 간다.
- 금융상품 설명은 필요 이상으로 수학적으로 쓰지 않는다.
- 독자가 `왜 이 사건이 중요했는지`를 한 장면씩 따라갈 수 있어야 한다.
- 구조 설명과 드라마틱한 타임라인을 번갈아 배치한다.
