## 이 폴더의 파일들

### 1. `black_monday_2024_book_outline_2026-04-18.md` — 목차
**쓰임**: 집필의 설계도. 챕터 순서, 각 챕터의 핵심 장면, 전체 서사 흐름을 담고 있다.

집필 시 이 파일을 먼저 열어 해당 챕터의 위치와 역할을 확인한다.
- 각 챕터가 전체 서사에서 어떤 기능을 하는지 (긴장 고조 / 반전 / 교훈)
- 앞뒤 챕터와의 연결 포인트
- 책의 핵심 테제와 해당 챕터의 관계

목차를 수정할 때는 전체 흐름과 파트 균형을 함께 고려한다.

---

### 2. `black_monday_2024_research_2026-04-18.md` — 리서치
**쓰임**: 사실 확인의 기준점. 날짜, 수치, 인물 발언, 출처를 정리한 작업 메모다.

집필 시 이 파일을 열어:
- 각 챕터에 필요한 정확한 수치와 날짜를 확인한다
- `확인 필요` 표시 항목은 챕터 집필 전에 웹 검색으로 재확인한다
- 새로 확인한 사실은 이 파일에 업데이트해 다음 챕터에서도 재사용한다
- 인물 발언은 반드시 출처를 표기한다 (날짜, 매체명)

**중요**: 이 파일의 내용은 대화 맥락에서 정리된 것으로, 일부 수치는 `확인 필요` 상태다. 본문에 수치를 쓰기 전 반드시 1차 소스를 재확인한다.
