# Dot Com Bubble Writing Folder

- purpose: `닷컴 버블`(1993~2003)을 중심으로 한 장기 집필 프로젝트의 원고 저장 폴더
- audience: 일반 개인투자자도 재미있게 읽을 수 있는 교양서/연재물/에세이 독자

## 이 폴더에 저장하는 것

- 책 전체 목차
- 서문, 프롤로그, 에필로그 초안
- 장별 시놉시스
- 본문 챕터 초안
- 문장 다듬기 전 단계의 작업 원고
- 구성 메모, 브리지 문단 메모, 장 전환 메모

## 이 폴더에 저장하지 않는 것

- raw article 원문 아카이브
- 기사 수집용 상태 파일
- 검색용 SQLite DB
- 단순 조사 로그

위 자료들은 `/Research/`와 `Research/research_archive.sqlite3`에서 관리한다.

## 프로젝트 범위

이 폴더는 아래 흐름을 하나의 긴 서사로 묶는 집필 프로젝트를 다룬다.

- 1993~1995: 모자이크·넷스케이프와 웹의 상업화
- 1995: 넷스케이프 IPO — 수익 없는 회사의 상장이 선례가 되다
- 1996~1998: 골드러시, 그린스펀의 경고, LTCM 위기와 연준 금리 인하
- 1999~2000.03: "아이볼"이 수익보다 중요했던 시절, Pets.com·Webvan·AOL합병, 나스닥 5,048
- 2000~2001: 바론스의 폭탄, 긴 하락, 통신 버블 붕괴, 9·11
- 2001~2003: 엔론·월드컴 회계 사기, 애널리스트 이해충돌 스캔들, 사베인스-옥슬리법
- 2020년대: AI 열풍 — 닷컴 버블과의 비교 (Part 6, 에필로그 이후 현재 시점 파트)

## 편집 원칙

- 설명은 쉽고 서사는 강하게 간다.
- 금융 개념 설명은 필요 이상으로 수학적으로 쓰지 않는다.
- 독자가 `왜 이 사건이 중요했는지`를 한 장면씩 따라갈 수 있어야 한다.
- 구조 설명과 드라마틱한 타임라인을 번갈아 배치한다.
- GFC·코로나 시대와 달리, 이 버블의 연료는 레버리지가 아니라 **내러티브**였다는 점을 중심 축으로 삼는다.
- 사기와 광기가 어떻게 공존했는지, 무엇이 진짜 혁신이고 무엇이 허상이었는지를 독자가 스스로 판단할 수 있도록 서술한다.

## 광기의 3부작 — 다른 프로젝트와의 관계

이 프로젝트는 GFC, COVID Era와 함께 **광기의 3부작**을 이루는 독립 단행본이다.

| 프로젝트 | 거품의 연료 | 핵심 충격 |
|---|---|---|
| **Dot Com Bubble** | 내러티브 ("새로운 경제") | 주식 단독 붕괴 + 회계 스캔들 |
| **Global Financial Crisis** | 레버리지 (MBS·CDO·CDS) | 신용 시스템 전체 붕괴 |
| **COVID Era** | 국가 주도 유동성 | 인플레이션 + 금리 쇼크 |

- 세 버블의 공통 주제(군중 심리, 규제 공백, 위험의 이동)가 어떻게 반복·변형되는지를 의식하며 집필한다.
- 각 책은 독립적으로 읽힐 수 있어야 하되, 함께 읽으면 더 큰 그림이 보이도록 설계한다.
