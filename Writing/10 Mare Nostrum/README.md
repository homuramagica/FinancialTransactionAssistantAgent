# Mare Nostrum Writing Folder

- purpose: `공화정 말기~제정 초기 로마 제국의 공급망`을 중심으로 한 장기 집필 프로젝트의 원고 저장 폴더
- audience: 경제사·세계사에 관심 있는 일반 독자, 공급망과 세계화의 기원에 흥미를 느끼는 교양 독자
- 집필 기간: 2026년 4월~

## 이 폴더에 저장하는 것

- 책 전체 목차
- 프롤로그, 에필로그 초안
- 장별 시놉시스
- 본문 챕터 초안
- 구성 메모, 브리지 문단 메모, 장 전환 메모

## 이 폴더에 저장하지 않는 것

- raw article 원문 아카이브
- 기사 수집용 상태 파일
- 검색용 SQLite DB
- 단순 조사 로그

위 자료들은 `/Research/`와 `Research/research_archive.sqlite3`에서 관리한다.

## 프로젝트 범위

이 폴더는 아래 흐름을 하나의 긴 서사로 묶는 집필 프로젝트를 다룬다.

- 기원전 30년, 아우구스투스의 알렉산드리아 입성 — 지중해가 처음으로 하나의 경제권으로 묶인 날
- 군단이 깔아놓은 도로와 요새 — 모든 공급망의 물리적 기반
- 오스티아 — 세계 최초의 대규모 물류 허브
- 풍요로운 식탁과 그 이면 — 이집트 밀이 이탈리아 자영농을 무너뜨린 과정
- 황남대총의 녹색 유리병 — 로마 글로벌 소싱의 도달 범위
- 파르티아라는 장벽 — 닿지 않는 곳과 중간상의 폭리
- 노예 공급망 — 제국의 진짜 에너지원
- 로마가 수출한 라이프스타일 — 목욕탕과 포룸이 만든 공통 문화 코드
- 바울의 지중해 — 물건과 사람 위에서 아이디어도 흘렀다
- 역병 — 연결이 전염병의 고속도로가 됐다
- 데나리우스의 죽음과 3세기 위기 — 공급망이 무너질 때
- 1453년, 콘스탄티노플 — 로마는 끝났는가, 네트워크는 끝났는가

## 편집 원칙

- 설명은 쉽고 서사는 강하게 간다.
- 역사 서술이지만 독자가 장면 속에 있는 것처럼 느껴야 한다.
- 트리비아와 세부 사실을 아낌없이 살린다 — 숫자, 이름, 유물, 기록.
- 각 챕터는 구체적인 장면이나 사물에서 시작해 큰 구조로 확장한다.
- 현대와의 연결 고리를 자연스럽게 삽입한다 — 독자가 "이게 지금도 똑같다"고 느끼게.
- 이 책의 중심 서사는 **공급망이 문명을 만든다**는 것이다 — 물건의 이동이 곧 문화의 이동이었다.
- 프롤로그(알렉산드리아 입성)와 에필로그(콘스탄티노플 입성)의 수미쌍관 구조를 끝까지 살린다.
