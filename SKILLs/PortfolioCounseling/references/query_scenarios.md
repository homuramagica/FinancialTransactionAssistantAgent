# 사용자 질의 시나리오 맵 (포트폴리오 상담)

목표:
- 포트폴리오 관련 질문을 빠르게 인텐트로 분류한다.
- 인텐트별 기본 출력 형식과 최소 데이터 요건을 고정한다.
- 모호한 요청도 가정 기반 1차 상담안으로 즉시 대응한다.

## 1) 빠른 진단형

## 9) 모호한 질문 대응 규칙
- 보유자산 미지정: 기본 ETF 유니버스로 1차 예시안을 먼저 제시한다.
- 기간 미지정: `1Y 데이터 + 향후 6~12개월 시나리오`를 기본으로 사용한다.
- 목표 미지정: `성장/균형/방어` 3안 동시 제시 후 선택하도록 유도한다.
- 출력 깊이 미지정: `Flash`부터 시작한다.
- `유지/축소/매도` 류 질문: `python3 scripts/world_memory_cli.py list --days 21 --format md`를 선조회하고, 답변 첫 문장에 `as of (KST)`를 명시한다.
