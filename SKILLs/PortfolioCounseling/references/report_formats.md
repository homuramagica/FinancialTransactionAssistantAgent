# 포트폴리오 상담 리포트 포맷 리서치 (웹 기반)

아래 포맷은 실제 상담 흐름에 맞게 재구성한 실행 규칙이다.
수식/이론 나열보다, 사용자 의사결정 문장과 실행 순서를 우선한다.

## 1) Friendly Portfolio Check (기본값)
추천 상황:
- 사용자가 스타일을 지정하지 않았을 때
- 현재 포트 상태를 빠르게 진단하고 싶을 때

필수 섹션:
1. 한 줄 결론 + as of(KST)
2. 현재 포트 진단(집중도/리스크/행동요인)
3. 즉시 실행 액션 3개
4. 3시나리오(Baseline/Upside/Downside)
5. 다음 점검 시점

작성 규칙:
- 어려운 수식 대신 쉬운 의미 설명을 먼저 제시한다.
- "왜 이 액션이 필요한가"를 한 줄로 붙인다.

## 2) Goal-based Allocation Plan
추천 상황:
- "은퇴/주택/교육비 목표에 맞춘 포트" 요청
- 장기 계획 중심 상담

필수 섹션:
1. 목표/기간/우선순위 정리
2. 목표별 버킷 분리
3. 자산 배분안(현재 vs 제안)
4. 단계별 리밸런싱 일정
5. 목표 달성 점검 지표

작성 규칙:
- 목표별로 리스크를 분리해 설명한다.
- 단일 포트 결론보다 목표별 실행순서를 강조한다.

## 3) Risk Budget & Tail Defense Memo
추천 상황:
- "하락장 대비", "리스크 줄이기" 요청
- 변동성/낙폭 관리가 핵심일 때

필수 섹션:
1. 리스크 기여도 분해
2. 집중 리스크(섹터/자산/스타일) 식별
3. 꼬리위험 방어 수단
4. 무효화 조건(방어 해제 기준)
5. 비상시 행동 규칙

작성 규칙:
- 리스크 축소가 수익기회 상실로 이어질 수 있음을 함께 고지한다.
- 방어 전략은 단계적으로 제시한다.

## 4) Lifecycle Coaching Note
추천 상황:
- "나이/생애주기 기반으로 조정" 요청
- 은퇴 시점이 명확한 경우

필수 섹션:
1. 현재 라이프사이클 단계
2. 단계별 위험노출 권장치
3. 전환 트리거(시간/시장/개인 사정)
4. 인출기 안정성 점검
5. 점검 주기

작성 규칙:
- 연령 자체보다 목표 시점과 현금흐름을 우선한다.
- 과도한 공격/과도한 방어를 동시에 경계한다.

## 5) Behavioral Coaching Brief
추천 상황:
- "멘탈이 흔들린다", "뉴스에 휘둘린다" 요청
- 손실 회피/FOMO/추격매수 패턴 교정 필요

필수 섹션:
1. 관찰된 편향 요약
2. 편향별 교정 규칙
3. 실수 재발 방지 체크리스트
4. 거래 전 질문 3개
5. 리뷰 루틴

작성 규칙:
- 판단을 비난하지 않는다.
- 행동 규칙은 짧고 반복 가능한 문장으로 작성한다.

## 6) 포맷 선택 규칙
- 사용자가 특정 형식을 지정하면 해당 포맷을 우선 적용한다.
- 지정이 없으면 `Friendly Portfolio Check`를 기본으로 사용한다.
- 초보자 요청이면 `SKILLs/Newbie.md` 우선 적용 후 단순 포맷으로 축약한다.
