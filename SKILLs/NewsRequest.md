# NewsRequest

## 적용 조건
- 사용자가 최신 뉴스/속보/시장 업데이트(“최신 뉴스”, “속보”, “시장 업데이트”, “breaking news” 등)를 요청한 경우 이 지침을 따른다.
- 답변과 정보 제공은 **한국어**로 한다.

## 데이터 소스 (항상 전부 사용)
- 텔레그램 3개 피드(금융 속보 FEED, 통합 CSV): `https://rss.app/feeds/_8HzGbLlZYpznFQ9I.csv`
- 정규 언론 FEED: `https://rss.app/feeds/_hc8HiU0HyBWHfWoM.csv`
- 텔레그램 원본 피드(직접 탐색):
  - `https://t.me/s/WalterBloomberg`
  - `https://t.me/s/FinancialJuice`
  - `https://t.me/firstsquaw` (필요 시 `https://t.me/s/firstsquaw`로 게시글 목록 확인)

## 통합 절차
1) 각 FEED를 확인해 최신 항목을 확보한다.
2) 시간은 **KST(Asia/Seoul)**로 변환한다.
3) 중복은 URL 우선으로 제거하고, 없으면 제목+시간 조합으로 제거한다.
4) 최신성/심각도에 가중치를 두고 정렬한다.
5) 각 소식에는 **출처명과 링크**를 반드시 제공한다.

## 보고서 출력 형식
- Markdown, List, Table 등을 적극적으로 사용한다.
- 하나의 긴 문장보다 **짧고 간단한 문장** 여러 개를 사용한다.
- 각 항목에는 필요한 지점에 출처를 `[출처 매체명](URL)` 형식으로 표기한다.
- 사용자가 `뉴스레터 스타일`, `산문`, `블룸버그 스타일`, `지면 확대`를 요청하면
  **산문형 뉴스레터 모드**를 우선 적용한다.
  - 권장 문단 수: **8~14개**
  - 문단당 권장 길이: **3~5문장**
  - 문단 구성: 사건 요약 → 시장 함의 → 확인할 트리거 → 출처

```
# YYYY월 MM월 DD일 hh:mm (KST) 주요 속보

## (emoji) (이슈별 제목)
- 짧은 문장으로 핵심 요약.
- 필요한 지점에 출처 표기: [출처 매체명](URL)
- 동일 이슈의 추가 포인트를 짧은 문장으로 나열.

## (emoji) (다음 이슈 제목)
- 동일 형식 반복.

## 오피니언 기사
- 매체명 - Original English Title / 한국어 번역 제목: 간단한 요약
- 동일 형식 반복.
```

## 오피니언 기사 처리
- URL에 `opinion` 문자열이 포함된 항목은 **오피니언 섹션**으로 분리한다.
- 오피니언 제목은 **한국어 번역**을 제공한다.
- 가능하면 간단한 요약을 함께 제공한다.

## 뉴스레터 산문 모드 템플릿 (요청 시)
```
# YYYY년 MM월 DD일 hh:mm (KST) 뉴스레터 브리핑

## 오프닝
(시장 뉴스 레짐 한 문장)

## 본문
(문단 1)
(문단 2)
...
(문단 8~14)

## News Tape (요약)
- HH:MM KST | 핵심 헤드라인 ([출처](URL))
...
```

- 단순 나열형을 피하고, 문단 사이 연결 문장을 넣어 흐름을 만든다.
- 동일 사건을 반복 인용하지 말고, 하나의 문단에서 통합 정리한다.

## 보고서 종료 문구 (필수)
아래 문장을 **그대로** 출력한다.

"이상이 현재 글로벌 시장의 속보입니다. 좀 더 자세히 알고 싶은 내용이 있으신가요? 
혹은, 조금 뒤에 다시 속보 업데이트를 요청하시면 각종 정보 플랫폼과 언론 출처 모두에서 새로 업데이트 된 정보만 추려서 다시 제공 드리도록 하겠습니다."

## 업데이트 요청 처리
- 사용자가 다시 속보 업데이트를 요청하면 **두 CSV FEED를 반드시 모두 재호출**한다.
- 텔레그램 원본 피드도 다시 확인한다.
- 기존에 제공한 소식은 **다시 제공하지 않는다**.
- 새 항목이 적으면 **간결하게** 새로운 소식만 제공한다.

## 심층 맥락 요청 시 (사용자가 요구할 때만)
- RSS에서 얻은 단서를 기반으로 **구체적인 영어 검색어**를 사용한다.
- 한국어 자료는 검색/인용하지 않는다.
- 신뢰도 낮은 출처(블로그/유튜브 등)는 배제한다.
- **허용 출처:** Reuters, Bloomberg, Financial Times, Wall Street Journal, Barron’s, Seeking Alpha, Benzinga
- 미국 중심의 글로벌 뉴스 맥락으로 보강한다.
- 보강 내용은 한국어로 요약하고, 출처 링크를 포함한다.
